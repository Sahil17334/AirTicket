## AirTicket
  REST APIs for Airticket Android App hosted at: https://github.com/rohanprasadofficial/Airticket-App 
### How to run
- Clone the repository
- Start mongodb server
- Add a env file
- Add into it 
  `EXPORT username="XXX"
   EXPORT password="XXX"
   EXPORT DBURL="XXX"
  `
- Run `source env`
- Add env file to .gitignore
- Run `npm i`
- Run `npm start`
